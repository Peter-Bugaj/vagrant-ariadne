Quick Start
-----------

    $ git clone https://github.com/myplanetdigital/ariadne.git
    $ cd ariadne
    $ git checkout `git describe --tags`
    $ bundle install
    $ librarian-chef install
    $ vagrant up
    $ cap deploy
